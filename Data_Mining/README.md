These projects were part of the "Data Mining" subject in the Master's in Data Science program at Bellevue University. 
Here is the description of each file:
- Solis-Toro_DSC550-T301_Week1.ipynb: Explores video game sales data through data cleaning, descriptive statistics, and publisher analysis with Pandas
- Solis-Toro_DSC550-T301_Week2.ipynb: Analyzes nutritional content across fast-food chains, identifying calorie density, protein-calorie correlations, and sodium levels.
- Solis-Toro_DSC550-T301_Week3.ipynb: Compares sentiment analysis methods (TextBlob vs. VADER) and prepares text data for machine learning with preprocessing and vectorization.
- Solis-Toro_DSC550-T301_Week4.ipynb: Builds regression models to predict car MPG using engine specs and weight, achieving 84% accuracy with feature importance analysis.
- Solis-Toro_DSC550-T301_Week5.ipynb: Compares Logistic Regression and Naive Bayes for sentiment analysis, achieving 88% accuracy with TF-IDF vectorization
- Solis-Toro_DSC550-T301_Week7.ipynb: Compares PCA and variance thresholding for housing price regression, and chi-square feature selection for mushroom classification, demonstrating trade-offs between dimensionality reduction and predictive performance.
- Solis-Toro_DSC550-T301_Week9.ipynb: Compares KNN, Logistic Regression, and Random Forest with grid search hyperparameter tuning for loan approval prediction. Random Forest achieved the best performance (83% accuracy) versus default KNN (71%), demonstrating ensemble methods' superiority for this classification task.
- Solis-Toro_DSC550-T301_TermProject.ipynb: Complete sleep quality prediction pipeline from exploratory data analysis through data preparation, including feature engineering, missing value handling, one-hot encoding, and feature scaling.
