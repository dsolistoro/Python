These projects were part of the "Statistics for Data Science" subject in the Master's in Data Science program at Bellevue University.
Here is the description of each file:
- Final-Projet_Solis-Toro_DSC_530-T301.ipynb: Analyzes workout tracker data to explore relationships between sleep, exercise metrics, and mood using statistical and regression modeling.
- SolisToro_530-T301_Week1.ipynb: Introduces core Python concepts including arithmetic operations, string manipulation, and the creation and modification of lists and tuples.
- SolisToro_530-T301_Week10.ipynb: Analyzes marijuana price trends and tests for serial correlation in transaction data using quadratic modeling and statistical testing
- SolisToro_530-T301_Week11.ipynb: Analyzes marriage survival trends using Kaplan-Meier and hazard estimation with bootstrap resampling.
- SolisToro_530-T301_Week3.ipynb: Analyzes pregnancy and birth data to compare first-born characteristics, such as gestational age and birth weight.
- SolisToro_530-T301_Week4.ipynb: Demonstrates probability distributions, survey sampling bias, and statistical calculations using real NSFG dataset.
- SolisToro_530-T301_Week5.ipynb: Analyzes income distribution using Pareto and normal models, calculating key statistics and skewness metrics.
- SolisToro_530-T301_Week7.ipynb: Compares statistical estimators, analyzes scatter plots and correlations, and simulates sampling distributions for inference.
- SolisToro_530-T301_Week8.ipynb: Investigates statistical power vs. sample size using t-tests and applies weighted linear regression to analyze BRFSS height-weight relationships.
- SolisToro_530-T301_Week9.ipynb: Uses regression to identify predictors of pregnancy length and applies data mining techniques to test factors affecting newborn sex ratio.
