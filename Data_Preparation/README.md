These projects were part of the "Data Preparation" subject in the Master's in Data Science program at Bellevue University. 
Here is the description of each file:
- Solis-Toro_DSC540-T301_Project-Milestone-2.ipynb: Cleans and transforms CDC public health data through standardization, outlier detection, and fuzzy matching for analysis.
- Solis-Toro_DSC540-T301_Project-Milestone-3.ipynb: Scrapes and cleans Wikipedia county income data, converting currency values and standardizing geographic names for analysis.
- Solis-Toro_DSC540-T301_Project-Milestone-4.ipynb: Pulls and cleans U.S. Census API data, creating an education-income dataset with outlier removal and ratio metrics.
- Solis-Toro_DSC540-T301_Project-Milestone-5.ipynb: Analyzes Florida county-level diabetes prevalence, identifies socioeconomic correlations, and creates a comprehensive, integrated dataset.
- Solis-Toro_DSC540-T301_Week1-2.ipynb: Demonstrates Python fundamentals, including list manipulation, text analysis, permutation generation, and CSV parsing.
- Solis-Toro_DSC540-T301_Week3-4.ipynb: Applies pandas for statistical analysis, web scraping for economic data, and SQLite for database management.
- Solis-Toro_DSC540-T301_Week5-6.ipynb: Cleans candy survey data through gender normalization, age grouping, and rating analysis, revealing top-rated treats and demographic preferences.
- Solis-Toro_DSC540-T301_Week7-8.ipynb: Scrapes Project Gutenberg's Top 100 eBook list, queries movie and nutrition APIs, and visualizes candy survey data through bar charts, histograms, and pie charts.
- Solis-Toro_DSC540-T301_Week9-10.ipynb: Executes SQL queries on a pet owner database to analyze demographic distributions and pet healthcare patterns.
